# Exercici 2 – Mètode genèric amb paràmetres diversos (Java 21)

## 📄 Description

This exercise introduces **generic methods** in Java and demonstrates how they allow flexible yet type-safe handling of multiple data types.

You will create a **generic method** capable of receiving and printing **three parameters of any type**, in any order. This highlights the power of generics compared to fixed-type implementations.

---

## 💻 Technologies Used

* Java 21 (LTS)
* Object-Oriented Programming (OOP)
* Java Generics
* Java Standard Library

---

## 📋 Requirements

To run this project, you will need:

* Java Development Kit (JDK) 21 (LTS)
